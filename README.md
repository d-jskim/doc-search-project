# 📊 데이터 현황 확인 프로그램
> 데이터 전처리 전 raw data의 데이터 구조, 카테고리 분포, 결측치 현황 파악, 통계량 계산 등 데이터 현황 확인

## 🚀 주요 기능
- **데이터 불러오기**: 주어진 경로의 파일을 불러와서 데이터프레임으로 반환
- **데이터 구조 확인**: 데이터 크기(행과 열의 수), 데이터 유형 확인
- **카테고리 분포 확인**: 카테고리별 문서 수와 평균 단어 수 확인
- **결측치 현황 파악**: 컬럼별 결측치 수, 결측치 비율(%) 및 심각도 확인
- **통계량 계산**: NumPy로 직접 계산 통계량 및 `describe()`통계량 비교
